Setup Firebase

Noteworthy is 100% serverless. To run it locally or on GitHub Pages, you need to connect your own Firebase project.
  1. Go to Firebase Console and create a project.
  2. Enable Email/Password Authentication under Build -> Authentication.
  3. Enable Cloud Firestore under Build -> Firestore Database.
  4. Create a Web App in your project settings, and paste the firebaseConfig JS object below.
